Anime Lovers UWU

Project By: Amitoj Singh, Jonathan Rodriguez, John Huynh;

The first is plot is quite simple. It is supposed to be a scatter, beeswarm, plot that plots the ratings of various animes. The size of nodes represents the number of episodes, the color represents the number of votes, and position represents the rating of anime. Interactibility between the nodes in the plot and the pie chart makes it so that a selection in the plot chart highlights the genres in the pie chart. The pie chart highlights an anime with genres, from there we see the average ratings in a genre, and the distribution of ratings for the selected anime.
